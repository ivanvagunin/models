# Hairdryer Fan

Replacement fan model for 3D printing.

## Files

- `hairdryer_fan.blend` - editable Blender source
- `hairdryer_fan_print_oriented.stl` - print-ready STL in the preferred print orientation

## Notes

- The center fit features were loosened slightly for FDM printing with a `0.4 mm` nozzle.
- The hex nut pocket was opened up a little to better fit a nut around `12 mm` across flats.
- The central round opening was opened up a little to better fit a hex screw head around `9 mm` across corners.
- This is still a mesh-based Blender source, not a parametric CAD model.

## Suggested Print Check

- If possible, test the center hardware fit first before committing to a full reprint.
